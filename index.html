Write a reward in plain English, an LLM turns it into a reward function, then an agent is trained from scratch on it — live, in your browser. Watch what the reward you wrote actually makes the agent do.

  How training works: your English is translated by an LLM (through your OpenRouter key) into a
  JavaScript function reward(prev, action, state) (shown above once generated). The agent is then
  trained from scratch with tabular Q-learning, all in your browser: it plays the game tens of thousands of times
  (a few seconds), starting with random moves (ε-greedy exploration, decaying) and updating a lookup table
  Q(state, action) ← Q + α·(r + γ·max Q(next) − Q) after every step.
  The video is the final greedy policy: from each state it takes the action with the highest learned Q-value.
  Nothing else is optimized — whatever you see, the reward you wrote caused it.
