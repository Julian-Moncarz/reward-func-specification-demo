  How training works: your English is translated by Claude into a Python function
  reward(prev, action, state) (shown above once generated). The agent is then trained from scratch with
  tabular Q-learning: it plays the game tens of thousands of times (a few seconds, locally), starting with random moves
  (ε-greedy exploration, decaying) and updating a lookup table
  Q(state, action) ← Q + α·(r + γ·max Q(next) − Q) after every step.
  The video is the final greedy policy: from each state it takes the action with the highest learned Q-value.
  Nothing else is optimized — whatever you see, the reward you wrote caused it.
